# 🚆 Delhi to Dehradun Express Route Finder

A simple and interactive web application that helps users explore the Delhi to Dehradun railway route, including major stations, train information, travel details, and route visualization.

## 📌 Project Overview

This project was developed using **HTML, CSS, and JavaScript** as part of my web development learning journey.

The website provides:

* Route information from Delhi to Dehradun
* Train details and travel insights
* User-friendly interface
* Responsive design
* Interactive route exploration

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## ✨ Features

* Clean and responsive UI
* Route visualization
* Station information display
* Train details section
* Interactive user experience
* Mobile-friendly design
